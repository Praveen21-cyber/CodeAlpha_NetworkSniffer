# Network Sniffer

## 📌 Description
This project is a basic network sniffer built using Python and Scapy. It captures network packets and displays important information such as source IP, destination IP, and protocol.

## 🚀 Features
- Captures network packets
- Displays source and destination IP addresses
- Shows protocol used

## 🛠️ Tools Used
- Python
- Scapy

## ▶️ How to Run
1. Install Scapy:
   pip install scapy

2. Run the program:
   python sniffer.py

## ⚠️ Note
This project may require Npcap for full packet capturing functionality on Windows.
